### Comandos Úteis

1. Inicialização do ambiente
```bash
$ source ./venv/bin/activate
```
2. Registro das actions criadas no arquivo `actions.py`
```bash
$ rasa run actions
```

3. Treinamento do modelo
```bash
$ rasa train
```

4. Inicialização do `rasa x` para visualização via UI
```bash
$ rasa x
```
